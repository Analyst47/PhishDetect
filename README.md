**PhishDetect** is a professional phishing URL analysis dashboard built with **HTML, CSS, and JavaScript**.  
It provides detailed URL classification, risk evaluation, and visual statistics ‚Äî designed to mimic a **SOC (Security Operations Center)** workflow and showcase cybersecurity skills.

---

## **Project Overview**

Phishing attacks are one of the most common cybersecurity threats, targeting users to steal credentials or sensitive data.  
This dashboard allows users to analyze URLs in real-time, classify their risk level, and understand why a URL may be considered safe, suspicious, or malicious.  

It is a fully **static web application** that runs entirely in the browser, requiring **no backend server**, making it simple to deploy and share via **GitHub Pages**.

---

## **Features**

- **Real-time URL classification:** Safe, Suspicious, or Malicious  
- **Heuristic analysis:** Highlights factors that influence classification (HTTPS, URL length, special characters, suspicious keywords)  
- **Interactive statistics:** Tracks total URLs analyzed, and counts for each category  
- **Dynamic charts:** Visualize URL distribution using **Chart.js**  
- **Professional cyber-themed design:** Neon, SOC-style layout for portfolio showcase  
- Fully static, **runs entirely in the browser**
